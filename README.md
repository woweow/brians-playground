# Brian's Playground

A modern, minimal personal website built with React, TypeScript, and Vite.

## Features

- Modern UI with Tailwind CSS v4
- Smooth animations with Framer Motion
- Accessible components with Radix UI
- State management with Zustand
- Fully responsive design

## Tech Stack

- React 18.3.1
- TypeScript 5.6.3
- Vite 6.x
- Tailwind CSS 4.x
- Radix UI
- Framer Motion
- Zustand

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments on every push.
